Source-Agnostic Knowledge Pipeline の Phase 2 では `wiki compile` を Export JSON のみ提供し、
プログラマティック呼び出し用の Compiler API は Phase 2.5 に送った（Q12-2 A 案採用）。
Phase 2 完了後、API 呼び出しのニーズが顕在化したタイミングで実装する。

## 備考

### スコープアウト理由
- Q12-2 A 案（implementation 最小主義）に準拠
- schema は最大主義のまま、implementation だけ scope cut している

  - CI/CD パイプラインへの組み込みが必要になった
  - 他プロジェクトからライブラリとして再利用したいケースが発生
